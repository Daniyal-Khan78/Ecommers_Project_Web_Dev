# ShopNest — Full-Stack E-Commerce Web Application

A complete e-commerce platform built with React JS (frontend) and Django REST Framework (backend).

## Technology Stack

| Layer | Technology |
|---|---|
| Frontend | React JS, CSS3, Fetch API, React Router |
| Backend | Django, Django REST Framework |
| Database | SQLite |
| Authentication | JWT (JSON Web Tokens) |
| Version Control | Git & GitHub |

## Features

### Mandatory
- User Authentication (Register, Login, Logout) with JWT
- Responsive Design (Mobile, Tablet, Desktop)
- CRUD Operations (Products, Categories, Orders)
- REST API Integration (React + Django)
- SQLite Database Connectivity
- Form Validation (Frontend + Backend)
- Navigation Routing (React Router)
- Error Handling (Error pages + API error handling)

### Optional
- Email Verification
- Stripe Payment Integration
- Charts & Analytics Dashboard
- Real-time Notifications
- Product Image File Uploading
- AI-based Product Recommendations

## Setup Instructions

### Backend (Django)
```bash
cd backend
venv\Scripts\activate        # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### Frontend (React)
```bash
cd frontend
npm install
npm start
```
